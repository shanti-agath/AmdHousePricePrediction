# AmdHousePricePrediction:
This project predicts the price of houses in Ahmedabad city using Machine Learning (Linear Regression model).

## Objectives
- Load and preprocess housing dataset
- Train regression model
- Evaluate model performance
- Predict prices
- Visualize results

## Technologies
- Python
- Pandas
- Scikit-Learn
- Matplotlib

## Steps Performed
- Load dataset
- Handle missing values
- Encode categorical features
- Split train/test data
- Train Linear Regression
- Predict prices
- Evaluate using MSE & R2
- Plot Actual vs Predicted Results
- Saved trained model

## Files
- ahmedabadHousing.csv  — dataset
- HousePricePrediction.ipynb — main notebook
- house_price_model.pkl — saved ML model

## How to Run
Upload notebook & CSV to Google Colab and run all cells.

## Result
Model is able to predict house prices based on available features.
